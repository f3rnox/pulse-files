# Pulse Files 1.0 Release Notes

Pulse Files is an Android file manager and storage cleaner built with Kotlin and Jetpack Compose.

## Highlights

- Browse device storage with a Material 3 file manager.
- Clean unnecessary files by reviewing junk, duplicates, large files, downloads, and APKs.
- Manage files and folders with copy, move, rename, delete, share, create-folder, and multi-select actions.
- Search and sort files, show hidden files, and remember preferred browsing behavior.
- Preview common media and text files directly in the app.
- Work with ZIP archives, including browsing, opening entries, extracting archives, and creating ZIP files.

## Cleaner

- Shows total, used, and free storage at a glance.
- Scans for temporary files, log files, cache files, duplicate files, large files, downloaded files, and APK installers.
- Lets users review suggested cleanup items before deleting.
- Supports ignoring large files so they do not keep appearing in cleanup suggestions.
- Provides settings for scan depth, cleanup categories, large-file threshold, and launch-time scanning.

## File Browser

- Supports folder navigation, breadcrumbs, pull-to-refresh, and back navigation.
- Includes name, size, and modified-date sorting.
- Supports local and recursive search.
- Opens images, audio, video, text, Markdown, and code files in built-in viewers/editors where supported.
- Opens other file types through Android system apps.

## Settings

- Configure default sort order, hidden-file visibility, recursive search, delete confirmation, and remembered folders.
- Choose the startup tab and optional start folder.
- Select light, dark, or system theme with dynamic color support.
- Review app version information from the About screen.

## Notes

- Requires Android 7.0 or later.
- Requires storage access permission to browse, scan, and manage shared device storage.
